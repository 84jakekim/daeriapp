이 앱은 카카오 지도 JavaScript API를 사용합니다. 무료로 키를 발급받을 수 있습니다.

  1. developers.kakao.com 가입/로그인
  2. 내 애플리케이션 → 애플리케이션 추가
  3. 앱 키 → JavaScript 키 복사
  4. 플랫폼 → Web 추가 → 사이트 도메인 등록 (로컬 테스트는 http://localhost)
